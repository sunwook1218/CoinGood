# CoinGood
CoinGood Project 스프링부트로 구현한 coin 가격정보 api적용 AWS에 배포 app연동

## 설계
AWS
MySQL(미정)
Server SpringBoot url에 대해 json만 응답   
-> 어플리케이션단에서 서버에서 response받은 정보를 자유롭게 표현할 수 있음.  
웹 페이지  React (FE개발자 도움)  
안드로이드 ViewPager로 SingleActiviy디자인 예정
